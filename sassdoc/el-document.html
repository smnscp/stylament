Unique Document Elements

These elements are unique to each document. Some of the selectors are not pure element selectors. But due to their uniqueness it makes sense to address them without any added class specificity.

:root

scss 
:root {
  font-size: var(--scale-root, 100%);
}

Document root. Equivalent to html.

:where(body, body > div) > footer:last-of-type

scss 
:where(body, body > div) > footer:last-of-type {
  // Clever sticky footer (https://css-tricks.com/a-clever-sticky-footer-technique/)
  position: sticky;
  inset-block-start: 100vh;

  // Spacing and color.
  padding-block: var(--size-8);
  color: var(--color-body-inverted);
  background: var(--color-grey-700);

  a:any-link {
    color: var(--color-links-inverted);
  }
}

The document’s main footer.
