Custom Properties

This is a set of global CSS custom properties constituting the DNA of the design system. They are generated from the global configuration and are used in the higher layers of the system.

Color

Color Palettes

scss 
:root {
  $conf: map.get(cfg.$merged, color);
  $prefix: "color";

  @each $name, $pal in map.get($conf, palettes) {
    @include color.palette(
      $name,
      map.get($pal, hue),
      map.get($pal, s-curve),
      map.get($pal, l-curve),
      map.get($pal, sl-curve),
      map.get($pal, range),
      $prefix
    );
  }

  @include prop-map.configured-properties($conf, $prefix);
}

Modular Font Size Scale

scss 
:root {
  $conf: map.get(cfg.$merged, typography, scale);
  $base: map.get($conf, base);
  $factor: map.get($conf, factor);
  $min: map.get($conf, min);
  $max: map.get($conf, max);
  $prefix: "scale";

  --#{$prefix}-factor: #{$factor};

  @for $i from $min through $max {
    --#{$prefix}-#{$i}: #{scale.modular($base, $factor, $i)};
  }

  --#{$prefix}-smaller: #{math.div(100%, $factor)};
  --#{$prefix}-larger: #{100% * $factor};

  @include prop-map.configured-properties($conf, $prefix);
}

Requires

$merged (Map)

Stroke Width Scale

scss 
:root {
  $conf: map.get(cfg.$merged, ui, stroke, width);
  $min: map.get($conf, min);
  $max: map.get($conf, max);
  $line: map.get($conf, line);
  $prefix: "stroke-width";

  @for $i from $min through $max {
    --#{$prefix}-#{$i}: #{line.$rlh * math.div($i, $line)};
  }

  @include prop-map.configured-properties($conf, $prefix);
}

A scale of stroke widths with $line units per root line.

Requires

$merged (Map)

$rlh (Number<rem>)
