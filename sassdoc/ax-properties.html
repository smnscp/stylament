Basic Custom Properties

This is a set of global custom CSS properties constituting the DNA of the design system. It was originally inspired by the well thought-through taxonomy of Pollen.

Color

Color Palettes

scss 
:root {
  $conf: map.get(cfg.$merged, color);
  $prefix: "color";

  @each $name, $pal in map.get($conf, palettes) {
    @include color.palette(
      $name,
      map.get($pal, hue),
      map.get($pal, s-curve),
      map.get($pal, l-curve),
      map.get($pal, range),
      $prefix
    );
  }

  @include prop-map.configured-properties($conf, $prefix);
}

Line Heights

scss 
:root {
  $scale-base: map.get(cfg.$merged, typography, scale, base);
  $scale-factor: map.get(cfg.$merged, typography, scale, factor);
  $scale-min: map.get(cfg.$merged, typography, scale, min);
  $scale-max: map.get(cfg.$merged, typography, scale, max);

  $line-min: map.get(cfg.$merged, typography, line, min);
  $line-root: map.get(cfg.$merged, typography, line, root);
  $line-sub: map.get(cfg.$merged, typography, line, sub);

  $base-unit: math.unit($scale-base);
  $scale-base-rem: if(
    $base-unit == "em",
    $scale-base * math.div(1rem, 1em),
    if($base-unit == "%", $scale-base * math.div(1rem, 100%), $scale-base)
  );
  $rlh: $line-root * $scale-base-rem !global;

  --line-root: #{$line-root};
  --line-root-abs: #{$rlh};

  @for $i from $scale-min through $scale-max {
    $font-size: scale.modular($scale-base, $scale-factor, $i);
    --line-#{$i}: #{rhythm.grid-line($font-size, $rlh, $line-min, $line-sub)};
  }
}

Line heights supporting the vertical rhythm at every step of the scale. Defines $rlh.

Modular Font Size Scale

scss 
:root {
  $conf: map.get(cfg.$merged, typography, scale);
  $base: map.get($conf, base);
  $factor: map.get($conf, factor);
  $min: map.get($conf, min);
  $max: map.get($conf, max);
  $prefix: "scale";

  --#{$prefix}-factor: #{$factor};

  @for $i from $min through $max {
    --#{$prefix}-#{$i}: #{scale.modular($base, $factor, $i)};
  }

  --#{$prefix}-smaller: #{math.div(100%, $factor)};
  --#{$prefix}-larger: #{100% * $factor};

  @include prop-map.configured-properties($conf, $prefix);
}

Stroke Width Scale

scss 
:root {
  $conf: map.get(cfg.$merged, ui, stroke, width);
  $min: map.get($conf, min);
  $max: map.get($conf, max);
  $line: map.get($conf, line);
  $prefix: "stroke-width";

  @for $i from $min through $max {
    --#{$prefix}-#{$i}: #{line.$rlh * math.div($i, $line)};
  }

  @include prop-map.configured-properties($conf, $prefix);
}

A scale of stroke widths with $line units per root line.
