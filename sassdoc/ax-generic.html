Generic CSS

These rules have very general selectors and hence apply to many elements. Together with custom properties they form the foundation of the design system.

The BasiCSS

Margin Reset

scss 
* {
  margin: 0;
}

Remove margins. Rely on flow spacing instead.

Convenient Box Sizing

scss 
*,
*::before,
*::after {
  box-sizing: border-box;
}

Set Box sizing default to border-box.

Target Scroll Spacing

scss 
:target {
  // Give URL fragment target enough air.
  scroll-margin-block-start: var(--length-3);
}

The current target element.

Abstract Coloring

scss 
* {
  ---clr-fg: var(---clr-pos);
  ---clr-bg: var(---clr-neg);
}

Set abstract colors.

Related

@mixin set-colors()

@mixin use-colors()

Coloring Mixins

@mixin set-colors()

Set abstract color properties to the color specified by $name and its counterpart (-contra).

Parameters

$name: (String)

Name of the color token.

$default: null (String)

Name of the fall-back color token.

Requires

$default (Map)

@mixin use-colors()

Use abstract color properties as color and background-color, respectively.

@mixin use-colors-inverse()

Use abstract color properties as background-color and color (inverse of use-colors), respectively.

Embedding

Default Embedding Boxes

scss 
:is(img, picture, video, audio),
:is(embed, iframe, object),
svg,
canvas {
  display: block;
  max-inline-size: 100%;
  block-size: auto;
}

Full-width Boxes

scss 
audio,
iframe:not([width]) {
  inline-size: 100%;
}

Flow Spacing

Use common block margin to establish consistency in flow layout and to support a baseline grid.

Before Headings

scss 
* + :is(h1, h2, h3, h4, h5, h6) {
  margin-block-start: var(--spacing-heading);
}

Before Sections

scss 
* + :is(article, section) {
  margin-block-start: var(--spacing-sectioning);
}

Before General Blocks

scss 
* + :is(p, blockquote, figure, figcaption, pre, dl, ol, ul, menu, dir, hr),
:is(h1, h2, h3, h4, h5, h6) + div,
:is(p, blockquote, figure, figcaption, pre, dl, ol, ul, menu, dir, hr) + div {
  margin-block-start: var(--spacing-block);
}

Not Before Nested Lists

scss 
li :is(ol, ul, menu, dir) {
  margin-block-start: unset;
}

Indentation

Nested Lists

scss 
li :is(ol, ul, menu, dir) {
  padding-inline-start: var(--spacing-inline);
}

Definition Details

scss 
dd {
  margin-inline-start: var(--spacing-inline);
}

Emphasized Blocks

scss 
blockquote,
figure {
  margin-inline: var(--spacing-inline);
}

Generic Typography

Root Line Height

scss 
html {
  line-height: var(--line-root);
}

The line-height of the root element.

Body Copy

scss 
body {
  font-family: var(--font-body);
  font-weight: var(--weight-body);
  font-size: var(--scale-0);
  line-height: var(--line-0);
}

Fonts for body copy.

Headings

scss 
:is(h1, h2, h3, h4, h5, h6) {
  font-family: var(--font-heading);
  font-weight: var(--weight-heading);
}

Fonts for headings.

h#{$level}

scss 
h#{$level} {
  font-size: var(--scale-#{4 - $level});
    line-height: var(--line-#{4 - $level});
}

Apply respective font-size and line-height to each heading level.

General Measure

scss 
:is(h1, h2, h3, h4, h5, h6),
:is(p, address, summary, figcaption, caption) {
  max-inline-size: var(--measure);
}

Establish a generic measure.

Monospaced

scss 
:is(code, kbd, samp),
pre {
  font-family: var(--font-mono);
}

Generic typography for monospaced elements.

Smaller

scss 
:is(small, sub, sup) {
  font-size: var(--scale-smaller);
}

Set scale-based font size.

Base Line Fix

scss 
:is(code, kbd, samp),
:is(small, sub, sup),
::marker {
  line-height: 0;
}

Prevent certain inline (pseudo) elements from affecting the line height.
