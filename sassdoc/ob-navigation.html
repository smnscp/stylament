Main Navigation Bar

Rather opinonated styling for the first nav.menu.

body > nav.menu:first-of-type

scss 
body > nav.menu:first-of-type {
  $size: var(--size-8);
  --toggle-face-size: #{$size};
  --toggle-face-padding: var(--size-1);
  --toggle-face-folded-background: white;
  --toggle-face-folded-radius: 0 0 var(--radius-lg);
  --toggle-face-folded-shadow: var(--elevation-5);

  & > label > .--toggle-face {
    background: white;
    box-shadow: var(--elevation-3);
  }

  & > menu {
    --item-height: #{$size};
    --item-line-height: var(--size-3);
    --item-padding: var(--size-3);

    box-shadow: var(--elevation-3);

    & > li {
      color: var(--color-body-inverted);
      background-color: var(--color-grey-700);
      transition: all 0.3s var(--easing-standard);
      &:hover {
        color: var(--color-grey-700);
        background-color: var(--color-body-inverted);
      }

      & > a {
        color: inherit;
        background-color: transparent;
        font-weight: var(--weight-bold);
        text-decoration: none;
      }
    }
  }
}

The first nav.menu is implicitly considered the main menu.

Navigation Menu

Common patterns for navigation menus. Separated into several aspects.

$nav-wide

scss 
$nav-wide: widths.$width-sm;

Breakpoint for switching to wide layout.

$size

scss 
$size: 2em;

Size factor of the menu.

nav.menu

scss 
nav.menu {
  position: relative;

  & .--toggle-state,
  & .--toggle-face {
    display: none;
  }
}

General navigation menu. This is the basic selector with the basic rules applied.

nav.menu.levelable

scss 
nav.menu.levelable {
  & > menu {
    display: block;

    & > li {
      display: block;
      text-align: start;
    }

    // Horizontal nav on wider screen.
    @media (min-width: $nav-wide) {
      display: flex;

      & > li {
        flex: 1;
        text-align: center;
      }
    }
  }
}

Levelable Nav. Places items on a horizontal level if enough space is available.

nav.menu.foldable

scss 
nav.menu.foldable {
  --toggle-face-size: #{$size};
  --inset-start: var(--toggle-face-size);
  --inset-end: 0;

  & > label {
    & > .--toggle-face {
      display: inline-block;
      position: absolute;
      inset-block-start: 0;
      inset-inline-start: 0;
      inline-size: var(--toggle-face-size);
      block-size: var(--toggle-face-size);
      padding: var(--toggle-face-padding);
      transition: all 0.3s var(--easing-standard);

      & > img {
        block-size: 100%;
      }
    }
  }

  & > menu {
    position: absolute;
    inset-block-start: 0;
    inset-inline-start: var(--inset-start);
    inset-inline-end: var(--inset-end);
    overflow: hidden;
    transition: inset-inline-end 0.3s var(--easing-standard);
  }

  @media (max-width: calc($nav-wide - 1px)) {
    // Backdrop when unfolded on small screen.
    & input.--toggle-state:checked ~ label {
      position: absolute;
      inset: 0;
      margin: 0;
      inline-size: 100vw;
      block-size: 100vh;
      background: rgba(0, 0, 0, 0.5);
    }

    & input.--toggle-state:not(:checked) {
      & ~ menu {
        inset-inline-end: 100%;
      }

      & ~ label > .--toggle-face {
        @include toggle-face-folded;
      }
    }
  }

  @media (min-width: $nav-wide) {
    & input.--toggle-state:checked {
      & ~ menu {
        inset-inline-end: 100%;
      }

      & ~ label > .--toggle-face {
        @include toggle-face-folded;
      }
    }
  }
}

Foldable Nav. Allows to toggle the display of the menu.

nav.menu.sticky

scss 
nav.menu.sticky {
  position: sticky;
  inset-block-start: 0;

  menu {
    --item-height: #{$size};
  }
}

Sticky Nav. Makes navbar stick to the top of the viewport.
